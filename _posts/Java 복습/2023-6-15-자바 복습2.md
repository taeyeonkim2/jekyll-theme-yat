## InputStream / OutputStream

-바이트 단위로 읽고 쓴다

-FileInputStream / FileOutputStream

```1=java
InputStream in=null;
	
		try {
			in=new FileInputStream("d:/kty/zz.txt"); //윈도우의 절대경로.
		} catch (FileNotFoundException e) {
			e.printStackTrace();
		}
		int data;
		while(true) {
			try {
				data=in.read();
				if(data==-1)break;
				System.out.println(data); //txt에 ABCDE를 65 66 67 유니코드로 읽어 들인다.
				
			} catch (IOException e) {
				e.printStackTrace();
			} finally {
				try {
					in.close();
				} catch (IOException e) {
					e.printStackTrace();
				}
			}	
		}
```
