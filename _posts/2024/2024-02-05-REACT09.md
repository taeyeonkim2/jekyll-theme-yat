```3=Header.js
const Header = () =>{

```

## 재활용 가능한 컴포넌트로 개선 Props

git checkout -f component/header-1

위에는 '검색'이 하드코딩되어있어서 나중에 재활용하기가 힘들잖아

```2=Header.js

const Header = (props) =>{
    return(
        <header>
            <h2 className="container">{props.title}</h2>
        </header>
    )
}

```

```3=App.js

export default class App extends React.Component{
    render(){
        return <Header title="검색"/>
    }
}

```
