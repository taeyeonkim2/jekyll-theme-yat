## Card

git checkout 1-think/10-order-page

컴포넌트화되기위해서는 자신이 갖고있는 로직, 상태를 버려야해. 하드코딩x

```2=Card.jsx

const Card = ({header,data=[],footer}) => (
    <div className="Card">
        <header>{header}</header>
        <main>
            {data.map(({term,description})=>(
                <dl key={term}>
                    <dt>{term}</dt>
                    <dd>{description}</dd>
                </dl>
            ))}
        </main>
        <footer>{footer}</footer>
    </div>
)

export default Card;

```
